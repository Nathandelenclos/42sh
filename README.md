# 42sh

In this project, you are asked to make a advanced shell based on the
example [tcsh](https://fr.wikipedia.org/wiki/Tcsh).
***

## How to use ?

### Installation CSFML

### Build

```bash
make
```

### Usage

```bash
./42sh
```
